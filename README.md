## 프로젝트 소개
- 식당 예약 서비스인 '캐치테이블' 서비스의 기능들을 클론코딩한다. 실제 운영되는 서비스를 직접 개발해보며 개발 실력을 향상하고 팀 협업 경험을 쌓는다.
